EPISODE 1: Computing beyond Buchholz's Hydra

Definitions

CBHS-1

For n < ω:

[n] = BH(n)
[n,0] = [n]
[n,1] = [[n,0],0]
[n,m] = [[n,m-1], m-1]
[n,m,0] = [n,m]
[n,m,q] = [[n,m,q-1], [n,m,q-1], q-1]

CBHS-2

Let Φ be a valid CBHS-1 array.

{Φ}0,0 = Φ
{Φ}1,0 = [Φ, Φ, Φ]
{Φ}n,0 = [{Φ}n-1,0, {Φ}n-1,0, {Φ}n-1,0]
{Φ}n,m = {Φ}{Φ}n,m-1, m-1}
