The Gevius List

Lucian V. — February 2026
Abstract. We introduce a hierarchy of rapidly growing constructions organized into three episodes: CBHS arrays, BEAF-style symbolic hyperstructures, and the Jupiterian Algorithm (JA₀).

1. Episode I: Computing Beyond Buchholz's Hydra

1.1 CBHS-1

For n < ω:

[n] = BH(n)
[n,0] = [n]
[n,1] = [[n,0],0]
[n,m] = [[n,m-1], m-1]
[n,m,0] = [n,m]
[n,m,q] = [[n,m,q-1], [n,m,q-1], q-1]

1.2 CBHS-2

Let Φ be a valid CBHS-1 array.

{Φ}0,0 = Φ
{Φ}1,0 = [Φ, Φ, Φ]
{Φ}n,0 = [{Φ}n-1,0, {Φ}n-1,0, {Φ}n-1,0]
{Φ}n,m = {Φ}{Φ}n,m-1, m-1}

3. Episode III: Jupiterian Algorithm (JA₀)

3.1 Nu-linear Level

ν([a,0,0]) = a↑a
ν([a,b,0]) = ν([a↑X, b-1, 0]), X = ν([a↑a, b-1, 0])
ν([a,b,c]) = ν([a↑X, b↑X, c-1]), X = ν([a↑a, b↑b, c-1])

3.2 Nu-dimensional Level

ν([a,b,c]⁽⁰⁾) = ν([a,b,c]#[a,b,c])
ν([a,b,c]⁽ⁿ⁾) = ν([ν([a,b,c]⁽ⁿ¯¹⁾), ν([a,b,c]⁽ⁿ¯¹⁾), ν([a,b,c]⁽ⁿ¯¹⁾)]⁽ⁿ¯¹⁾)

Γ⁰₀(n) = ν([n,n,n]⁽ⁿ⁾)
